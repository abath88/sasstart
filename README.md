## Sassart is a simply program wich create empty sass project with 7-1 pattern. Sassstart created directory three:
+ +project_name
  - -index.html
  + +css
  + +js
  + +img
  + +sass
    - -main.scss
    + +abstracts
      - -_variables.scss
      - -_functions.scss
      - -_mixins.scss
      - -_placeholders.scss
    + +base
      - -_base.scss
      - -_animations.scss
      - -_typography.scss
      - -_utilities.scss
    + +components
    + +layout
    + +pages
    + +themes
    + +vendors

## Some from this files are created with basic configuration.
